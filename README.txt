INFO:
The front end of the code is written in Angualar
The backend is written in Node.js
Express.js framework is used in Node.js
The db used is MongoDB (connected to an online mongodB database from mlab.com)

CODE STRUCTURE:
There are 2 folders front and back. Front folder contains the angular code and the back folders contains the node.js code. The node_modules folder is removed from both these folders.

HOW TO RUN ONLINE:
- Working link: https://kitchen-display-application.herokuapp.com/

HOW TO RUN LOCALLY:
-go to "back" folder.
-run "npm install"
-run "npm start"
-access it on "http://localhost:3000"
